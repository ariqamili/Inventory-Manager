# Inventory Management System

A full-stack inventory management system built with Laravel and PostgreSQL for tracking products, stock levels, categories, and warehouse operations.

---

## Features

- Product management (CRUD operations)
- Category-based organization of inventory
- Stock tracking with automatic quantity updates
- Low-stock monitoring system
- Supplier and warehouse structure (if applicable)
- Reporting for inventory status and stock movement
- Secure backend architecture using Laravel MVC

---

## Tech Stack

- Laravel (PHP)
- PostgreSQL
- Blade Templates
- MVC Architecture

---

## Project Structure

- `app/Models` – Database models
- `app/Http/Controllers` – Business logic
- `resources/views` – UI templates
- `routes/web.php` – Application routes
- `database/migrations` – Schema definition

---

## Architecture

The system follows the MVC pattern, separating logic, database structure, and UI for maintainability and scalability.

---

## Highlights

- Efficient stock tracking system
- Clean relational database design
- Scalable Laravel backend structure
